Our Project Details

In this Project, we have implemented two papers and used their results to make other fancy implementations. For example,
Similar Movies: Where the user gives the movie name, and then we provide a list of similar movies.
Hybrid Recommendation: In this, we provide the user and a movie as input and then recommend the film to him. This is done with the help of MLP implemented for the Neural Collaborative Filtering paper.
Content-Based: In this, we provide a genre to the software, and it recommends the movies based on the TransFM model
